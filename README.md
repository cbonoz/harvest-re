Harvest predictions
---

Finds good value properties based off realtor.com data

### How to run

#### Client

<pre>
cd ./harvest-ai
yarn
yarn dev
</pre>

The app should now be running on port 3000.

#### Server
<pre>
  pip install -r
  uvicorn main:app --reload
</pre>
